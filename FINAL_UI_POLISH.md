## Структура SVG иконок

├── print.svg       (16x16) - Печать

## CSS классы для иконок

```css
.btn-icon {
    width: 16px;
    height: 16px;
    display: inline-block;
    vertical-align: middle;
    margin-right: 6px;
}

.success-icon-svg {
    width: 80px;
    height: 80px;
    margin-bottom: 20px;
}
```

## Использование в HTML

```html
<!-- Кнопка с иконкой -->
<button class="btn-secondary" onclick="downloadQR()">
    <img src="svgg/download.svg" alt="Download" class="btn-icon"> Скачать
</button>

<!-- Большая иконка успеха -->
<img src="svgg/success.svg" alt="Success" class="success-icon-svg">
```

## Преимущества SVG иконок

1. **Масштабируемость** - четкие на любом разрешении
2. **Цвет** - используют `currentColor`, адаптируются к цвету текста
3. **Размер** - легче чем эмодзи
4. **Кроссплатформенность** - одинаково выглядят везде
5. **Доступность** - можно добавить alt текст
